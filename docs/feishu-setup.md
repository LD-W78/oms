# 飞书配置指南

## 1. 创建飞书应用

### 步骤：
1. 访问 [飞书开放平台](https://open.feishu.cn/)
2. 登录飞书账号
3. 点击"创建企业自建应用"
4. 填写应用名称和描述
5. 上传应用图标

## 2. 配置应用权限

在应用管理页面，需要开通以下权限：
- `bitable:app` - 读取和写入多维表格
- `contact:user.base:readonly` - 读取用户信息
- `im:message:send_as_bot` - 发送消息

## 3. 创建多维表格

创建以下数据表：

### 订单表 (orders)
| 字段名 | 字段类型 | 说明 |
|--------|----------|------|
| 订单号 | 单行文本 | 唯一标识 |
| 客户 | 单行文本 | 客户名称 |
| 产品 | 单行文本 | 产品名称 |
| 数量 | 数字 | 订单数量 |
| 金额 | 数字 | 订单金额 |
| 状态 | 单选 | 待处理/处理中/已完成 |
| 日期 | 日期 | 订单日期 |

### 客户表 (customers)
| 字段名 | 字段类型 | 说明 |
|--------|----------|------|
| 客户名称 | 单行文本 | 公司名称 |
| 联系人 | 单行文本 | 联系人姓名 |
| 电话 | 电话号码 | 联系电话 |
| 邮箱 | 邮箱 | 电子邮箱 |
| 国家 | 单行文本 | 所在国家 |
| 客户等级 | 单选 | A级/B级/C级 |
| 状态 | 单选 | 活跃/非活跃 |

### 产品表 (products)
| 字段名 | 字段类型 | 说明 |
|--------|----------|------|
| 产品名称 | 单行文本 | 产品名称 |
| 类别 | 单行文本 | 产品分类 |
| SKU | 单行文本 | 产品编码 |
| 单价 | 数字 | 产品价格 |
| 库存 | 数字 | 库存数量 |

## 4. 配置环境变量

获取以下信息并填入 `.env.local`：

1. **App ID 和 App Secret**
   - 在应用详情页的"凭证与权限"页面
   - 复制 App ID 和 App Secret

2. **App Token**
   - 在多维表格详情页URL中
   - 格式: `https://xxx.feishu.cn/base/xxxxxxxxxx`
   - `xxxxxxxxxx` 就是 App Token

3. **表ID**
   - 在多维表格中打开表
   - 复制浏览器地址栏中的表ID

## 5. 发布应用

1. 在应用管理页面点击"版本管理与发布"
2. 创建新版本
3. 填写版本信息
4. 提交审核
5. 审核通过后发布

## 6. 测试连接

部署完成后：
1. 登录OMS系统
2. 访问"系统管理"-"Schema同步"
3. 点击"同步"按钮
4. 确认数据能正常读取
