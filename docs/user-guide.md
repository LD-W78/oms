# OMS订单管理系统 - 用户操作手册

## 登录

1. 打开系统首页
2. 输入用户名和密码
3. 点击"登录"

默认账号：
- 管理员: admin / 123456
- 经理: manager / 123456
- 销售: sales / 123456
- 查看: viewer / 123456

## 仪表盘

展示业务概览：
- 本月订单数
- 本月销售额
- 活跃客户数
- 待处理订单

## 订单管理

### 查看订单
1. 点击左侧菜单"订单管理"
2. 查看订单列表
3. 使用搜索和筛选功能

### 新建订单
1. 点击"新建订单"按钮
2. 填写订单信息
3. 点击"保存"

### 编辑订单
1. 在列表中找到目标订单
2. 点击"编辑"按钮
3. 修改信息后保存

## 客户管理

管理客户信息：
- 查看客户列表
- 添加/编辑客户
- 查看客户订单历史

## 权限说明

| 角色 | 订单 | 客户 | 产品 | 报表 |
|------|------|------|------|------|
| 管理员 | 全部 | 全部 | 全部 | 全部 |
| 经理 | CRUD | CRUD | CRUD | 读取 |
| 销售 | CRUD | CRUD | 读取 | 读取 |
| 查看 | 读取 | 读取 | 读取 | - |

## 离线使用

系统支持离线查看缓存数据：
- 网络断开时显示离线状态
- 可以查看之前加载的数据
- 重新联网后自动同步

## 常见问题

Q: 数据加载慢？
A: 检查网络连接，或联系管理员优化查询

Q: 无法登录？
A: 确认账号密码是否正确，或联系管理员重置
